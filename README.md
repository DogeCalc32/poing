A classic game of pong, or is it?
Poing is pong but you are the ball, trying to avoid the paddle.
It is for the TI-nspire cx ii.
To install it on your calc, transfer the .tns file with this website https://nspireconnect.ti.com/nsc/   (it only works on a google web browser)
And yes I know the score can go negative, it's intentional.
thank you for downloading this game!
Controls:
settings           = esc
change y direction = arrows
